Class TStringArray

DescriptionHierarchyFieldsMethodsProperties

Unit

StringsExt

Declaration

type TStringArray = class(TObject)

Description

This class implements methods and functions to acccess easily a dynamic array of Strings

You can think of it as the poor-man's version of TStrings with just the Strings–no Objects, no StringsAdapter, etc. It isn't even thread-safe, so be warned.
